# Fake News Generator | Python

A Python-based program that generates random fake news headlines using predefined subjects, actions, and locations.

## Features
- Random headline generation
- Uses predefined datasets for subjects, actions, and places
- Generates different output on each run

## Concepts Used
- Python lists
- Random module
- String manipulation
- Loops
